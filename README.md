## Clone
```
git clone https://github.com/emekadefirst/PropBlog-Class-Teachin.git blog

```

## Change directory to the clone project

```
cd blog

```

## Install dependencies

```
uv sync --frozen

```

## Activate virtual env 

# bash terminal

```
source .venv/Scripts/activate

```

# cmd

```

 .venv\Scripts\activate

```
